# enum-utils

Utilities for working with TypeScript enums and string unions.

## Requirements
- fromString(enumObj, str) -> T | undefined
- values(enumObj) -> T[]
- keys(enumObj) -> string[]
- isEnumValue(enumObj, val) -> type guard
- parseEnum(enumObj, str) throws if invalid
